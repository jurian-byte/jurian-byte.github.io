Reglas de negocio

Restricciones

RX1. No se maneja el carácter “ñ” en ningún campo de formulario, se validan los formularios al momento de los envíos para verificar que este carácter no se encuentre y en caso de que exista, se evitará el envío.

RX2. Para los registros de venta hechos por los clientes, la fecha en la que se realice dicha venta se añadirá al registro de manera automática.

RX3. El sistema crea un archivo xml en caso de que no exista uno en el sistema cuando se realice una acción de registro.

RX4. Si no existe un archivo PDF, se crea de manera automática, pero si este ya existe se sobrescribe con los datos actualizados.

RX5. Cada vez que se elimine un registro de cualquier archivo xml, los identificadores de cada registro se actualizarán para mantener la persistencia de los datos.
