Reglas de negocio

Requerimientos no funcionales

RNF1. La interfaz de usuario será implementada para navegadores web únicamente con HTML5 y CSS3.

RNF2. Los registros son almacenados en minúsculas y mayúsculas en los archivos xml.

RFN3. Todos los datos modificados en los archivos xml deben ser actualizados para el administrador.

RNF4. El sistema será codificado en el ambiente de desarrollo Netbeans en su versión 8.2 en un ambiente operativo Windows 10.

RNF5. El sistema debe desarrollarse aplicando el paradigma de diseño orientado a objetos.

RNF6. El sistema debe poseer interfaces gráficas simples.

RNF7. El sistema maneja archivos xml manteniendo la persistencia de los datos en todo momento.

RNF8. Los identificadores únicos de cada registro en la base de datos no serán visualizados en las interfaces gráficas de cliente.
