Descripción del proyecto

Visión del producto

Doña cuquita es una mujer emprendedora que se ha dedicado a la venta por catálogo de diversos productos, ella necesita un sistema informático con interfaz web que le permita mostrar un catálogo por el cual se le permitirá promocionar sus productos.

Prespectiva del producto

Se proyecta implementar un sistema web que permite almacenar información de los productos como son, precio, descripción y de manera opcional al menos una imagen del producto. El sistema web a implementar será independiente ya que no tendrá relación con otros sistemas.

Funciones del producto

El sistema permite:

  • Registrar clientes.
  • Añadir información.
  • Consultar información.
  • Ordenar información.
  • Visualizar productos.
  • Visualizar ordenes.
    • Objetivos del proyecto

    Objetivo general: Desarrollar un sistema de software que proporcione toda la información sobre los productos en venta

    Objetivos específicos:
  • Proporcionar una plataforma cómoda e intuitiva para los clientes de doña Cuquita.
  • Almacenar en un solo sistema la información de los productos de catálogo.
  • Facilitar la búsqueda de los productos e información de los mismos.
  • Proporcionar una herramienta que ayude a gestionar dicha información.
